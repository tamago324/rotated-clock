# Rotated Clock - 90° Shifted Time

時間を大切にするための90度回転時計。

## 制作動機

午前中に早く起きても、なぜか午前中の時間をだらだらと過ごしてしまうことがありました。

9時が12時の位置に表示されることで「もう12時だ！」という感覚になり、午前中の時間をより大切に使えるのではないかと考えて作りました。

## 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Nunito)
